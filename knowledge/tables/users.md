# Таблица `analytics.users`

> ⚠️ ШАБЛОН (синтетическая схема). Замени на реальную структуру.

**Назначение:** справочник пользователей и их атрибутов.

**Движок:** `MergeTree`, ключ сортировки `user_id`.

## Поля

| Поле | Тип | Описание |
|------|-----|----------|
| `user_id` | `UInt64` | Уникальный ID пользователя (первичный) |
| `signup_date` | `Date` | Дата регистрации |
| `country` | `LowCardinality(String)` | Страна (ISO-код) |
| `plan` | `LowCardinality(String)` | Тарифный план |

## Категориальные значения

- `country`: `RU`, `US`, `DE`, `FR`, `BR`
- `plan`: `free`, `pro`, `enterprise`

## Типичные джойны

- `users.user_id = events.user_id` — события пользователя
